##Fetch API Product Explorer

##📌Description
This project demonstrates how to use the JavaScript Fetch API to retrieve product data from an API and display it dynamically on a web page.

##🚀 Features
Fetches product data using the Fetch API
Displays product name, image, price, and description
Handles API responses using JavaScript
Simple and beginner-friendly project

##🛠️ Technologies Used
HTML5
Tailwind CSS
JavaScript (Fetch API)
